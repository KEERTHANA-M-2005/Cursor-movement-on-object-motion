Read this before you start

To ensure the smooth functioning of the Virtual Mouse, please check if your system meets the following requirements:

Computer System

  • Processor (CPU): Modern multi-core processor (Intel i5/i7 or equivalent)
  • RAM: At least 4GB (8GB recommended)
  • Graphics Card (GPU): Optional but helpful (e.g., Nvidia GTX)
  • Storage: 10GB available space (SSD or HDD)
  • Operating System: Windows, macOS, or Linux

How to Run this!

  • Step 1: Go to the Download page and click on the "Download" button.
  • Step 2: A ZIP file will be downloaded to your system.
  • Step 3: Extract (Unzip) the downloaded ZIP file.
  • Step 4: Inside the extracted folder, locate and run the installer.bat file.
  • Step 5: It will automatically install all the required dependencies and packages.
  • Step 6: After installation, navigate to the dist folder.
  • Step 7: Open the Geston App from the dist folder.
  • Final Step: You are now all set to operate Geston!

Input Devices

  • Keyboard & Mouse: Required for initial setup only
